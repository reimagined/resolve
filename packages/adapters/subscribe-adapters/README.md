# **Subscribe Adapters**
This folder contains `resolve-redux` adapters used to subscribe to events via [Socket.io](https://socket.io/).

Available ways:
* [resolve-subscribe-mqtt](./resolve-subscribe-mqtt)  
* [resolve-subscribe-socket.io](./resolve-bus-socket.io)
