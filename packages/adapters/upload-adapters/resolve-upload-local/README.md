# **resolve-upload-local**

This package is local uploader adapter.

#### Config
```js
uploadAdapter: {
    module: 'resolve-upload-local',
    options: {
      launchServer: true,
      protocol: 'http',
      host: 'localhost',
      port: 3001,
      directory: 'data',
      bucket: 'files',
      secretKey: 'key'
    }
  }
```
