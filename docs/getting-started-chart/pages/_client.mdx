**Client**

ReSolve comes with several client libraries that you can use on the frontend to communicate with a reSolve server. The following libraries are available:

<Tabs>
<TabItem value="client" label="@resolve-js/client" default>

A library that contains plain JavaScript functions used to communicate with a reSolve server. The code sample below illustrates the basic functionlity of tis library.

```js
const sendMessage = (userName, message) =>
  client.command(
    {
      aggregateName: 'Chat',
      type: 'postMessage',
      aggregateId: chatRoom,
      payload: {
        userName,
        message,
      },
    },
    (err) => {
      if (err) {
        console.warn(`Error while sending command: ${err}`)
      }
    }
  )
```

</TabItem>
<TabItem value="redux" label="@resolve-js/redux">

A library used to connect a React + Redux component to reSolve. The code sample below illustrates the basic functionlity of tis library.

```js
const { execute: executeCreateShoppingItem } = useReduxCommand((text) => ({
  type: 'createShoppingItem',
  aggregateId,
  aggregateName: 'ShoppingList',
  payload: {
    text,
    id: Date.now().toString(),
  },
}))
...
const { connect, dispose, selector: thisList } = useReduxViewModel({
  name: 'shoppingList',
  aggregateIds: [aggregateId],
  args: undefined,
})
```

</TabItem>
<TabItem value="reactHooks" label="@resolve-js/react-hooks">

A hook-based library used to connect React components to reSolve. The code sample below illustrates the basic functionlity of tis library.

```js
const getMyAggregates = useQuery(
  { name: 'MyAggregateList', resolver: 'all', args: {} },
  (error, result) => {
    setAggregates(result.data)
  }
)
...
const createAggregate = useCommand(
  {
    type: 'create',
    aggregateId: uuid(),
    aggregateName: 'MyAggregate',
    payload: { name: `MyAggregate ${aggregates.length}` },
  },
  (error, result, { aggregateId }) => {
    setAggregates([
      ...aggregates,
      {
        id: aggregateId,
        name: `MyAggregate ${aggregates.length}`,
        items: [],
      },
    ])
  }
)
```

</TabItem>
</Tabs>

You can also dirrectly use the HTTP expose by a reSolve server:

```js
const response = await fetch(`/api/query/ShoppingLists/all`, {
  headers: {
    'Content-Type': 'application/json',
  },
  method: 'GET',
})

const result = await response.json()
```
